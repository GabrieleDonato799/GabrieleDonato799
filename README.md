## About this repo

Optional coursework projects from my BA showcasing hands-on learning with different technologies.
  
### DoS (Donato's Server) - Computer Networks
A plain C HTTP parser and webserver on Linux, mostly checked with Valgrind. Experimented with path traversal blocking using a combination of realpath() filesystem path validation and chroot() & chdir() syscalls.

### AFSE (Album di Figurine dei Super Eroi) - Web & Mobile Programming
A mobile-first and responsive Bootstrap 5 website backed by Node.js, EJS templates and MongoDB to collect, buy and exchange superhero figures. Authenticating users with JWS tokens.
